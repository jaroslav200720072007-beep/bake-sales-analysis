# 🚲 Аналіз ефективності продажів мережі BikeStores (SQL, Excel, Tableau)

##  1. Передумови та огляд проєкту
Цей проєкт присвячений дослідженню операційної ефективності американської мережі магазинів велосипедів за період 2016–2018 рр. На основі аналізу транзакційних даних було розроблено стратегію оптимізації товарних запасів та сегментації клієнтської бази.

## Бізнес-проблема та цілі
Основний виклик для **BikeStores** — низький рівень утримання клієнтів (**Retention Rate = 15%**) та нерівномірний розподіл продажів між регіонами.

### Цілі аналізу
1.) **Сезонність (Seasonality)**  
   Визначити періоди пікового навантаження для оптимізації запасів, планування поставок та підготовки персоналу до сезонних змін попиту.

2.) **Асортимент та прибутковість**  
Проаналізувати, які категорії товарів і бренди приносять найбільше прибутку, щоб зрозуміти, на чому компанія заробляє найбільше.

3.) **Аналіз клієнтів**  
   Виділити сегмент VIP-клієнтів для впровадження програм лояльності, підвищення рівня повторних покупок та покращення Retention Rate.

## 4. Короткий виклад

### Огляд результатів
У проєкті я провів повний цикл аналізу — від написання SQL-запитів до побудови інтерактивного дашборду Після стабільного зростання у 2016 році, продажі компанії BikeStores досягли свого піку у 2017 році, продемонструвавши приріст виторгу на **66.4%**. Основні показники ефективності (KPI) демонструють високу концентрацію ринку в штаті Нью-Йорк та домінування спортивних категорій товарів. Хоча 2018 рік показав певне зниження обсягів, весняний сезон залишається головним періодом для прибутку, саме в цей період продажі є найвищими.

Нижче наведено огляд головної сторінки дашборду Tableau. Повний інтерактивний звіт доступний за посиланням у розділі технічної документації.

### Ключовий Показник Ефективності (KPIs):
*   **Total Revenue**: **$8.51M** — Загальний виторг за весь період (2016–2018).
*   **Total Units Sold**: **7,078** — Загальна кількість проданих велосипедів.
*   **Customer Base**: **1,445** — Кількість унікальних клієнтів.
*   **Retention Rate**: **15.19%** — клієнти, що повернулися за повторною покупкою.

### Тенденції продажів:
*   **Peak Performance**: Продажі компанії досягли максимуму в **квітні 2018 року**, що стало рекордним місяцем за весь період спостережень.
*   **Regional Dominance**: Штат **Нью-Йорк (NY)** є ключовим регіоном, генеруючи **65.74%** усього виторгу мережі через магазин у м. Болдвін.
*   **Category Leaders**: Категорія **Mountain Bikes** забезпечує майже половину доходу компанії (**$1.33M**).
*   **Brand Strength**: Бренди **Trek** та **Electra** сукупно контролюють понад **68%** ринку продажів мережі.

## 5. Глибоке занурення у висновки (Deep Dive)

У цьому розділі я розкриваю логіку аналізу: від виявлення бізнес-проблеми до її технічного підтвердження за допомогою SQL та візуалізації в Tableau.

### А. Сезонність та пікові навантаження
**Проблема:** Необхідність точного прогнозування складських запасів для запобігання дефіциту під час сплесків активності.
**Аналіз:** Вивчення продажів за 2016–2018 роки показало, що березень та квітень є критично важливими місяцями. Особливу увагу привертає квітень зафіксовано аномальний пік продажів.

*(На графіку **Revenue / Month**)*.

*   **Технічне підтвердження:** Для аналізу сезонності використано SQL-агрегацію по місяцях. Повний SQL скрипт[SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
SELECT 
    YEAR(order_date) AS year,
    MONTH(order_date) AS month,
    SUM(revenue) AS total_revenue
FROM "BikeStores.xlsx - Data"
GROUP BY year, month
ORDER BY year, month;
```
### Б. Продуктова стратегія та лідери ринку

**Проблема:** Визначення ключових сегментів для фокусування маркетингових зусиль та оптимізації складських запасів на основі прибутковості категорій.

**Аналіз:** Дані візуалізації підтверджують, що категорія **Mountain Bikes** є головним драйвером прибутку, забезпечуючи **48,97%** усього виторгу ($1,32 млн). Наступними за важливістю сегментами є **Cruisers** та **Cyclocross**, які сукупно дають майже **30%** доходу. Найнижчу ефективність демонструють **дитячі велосипеди** (лише **3,62%**), що вказує на низьку рентабельність та доцільність перегляду цього напрямку..

*   **Візуалізація:** 
    *(Дивлячись на дерево подібну діаграму **Revenue / Category Name** на скріншоті , де відображено домінування топових сегментів).*
    <img width="2940" height="1912" alt="Снимок 01 05 2026 в 16 16" src="https://github.com/user-attachments/assets/f5a4b9ab-9bf3-4801-86b7-7816e55d63dc" />

*   **Технічне підтвердження:** Для виділення ТОП-3 товарів у кожній категорії я застосував віконну функцію `RANK()`. Повний SQL скрипт[SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
WITH top_3 AS (
    SELECT category_name, product_name, total_units,
    RANK() OVER (PARTITION BY category_name ORDER BY total_units DESC) AS rnk
    FROM "BikeStores.xlsx - Data"
)
SELECT category_name, product_name, total_units
FROM top_3 
WHERE rnk <= 3;
```
### В. Аналіз ключових клієнтів (Top Customers)

**Проблема:** Необхідність ідентифікації найбільш цінних клієнтів для розробки стратегій персоналізованого маркетингу та VIP-обслуговування з метою підвищення загальної лояльності.

**Аналіз:** Візуалізація демонструє значну концентрацію виторгу серед топових покупців. Лідери рейтингу (Elinore Aguilar, Sharyn Hopkins) мають сумарний чек понад **$10,000–$11,000** кожен. Це підтверджує, що невелика група лояльних клієнтів формує значну частку високого чека, що є критично важливим на фоні загального показника Retention у **15.19%**.


*   **Візуалізація:**
    *(Стовпчаста діаграма **Top Customers** на скріншоті, яка ілюструє концентрацію великих чеків серед окремих лояльних покупців).*

*   **Технічне підтвердження:** Для аналізу використано SQL-запит, що визначає ТОП-10 клієнтів за сумою витрат. Повний SQL скрипт[SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
SELECT  customers, 
ROUND(SUM(revenue), 2) AS costs

GROUP BY customers

LIMIT 10
```

## 6. Стратегічні рекомендації

На основі виявлених висновків було сформовано наступні рекомендації, що мають на меті підвищення прибутковості та ефективності BikeStores:

*   **Оптимізація товарного портфеля для максимізації доходу.** Оскільки бренди **Trek** та **Electra** генерують понад **68%** усього виторгу, а категорія **Mountain Bikes** приносить майже половину прибутку (**$1.33M**), фокусування на цих сегментах є пріоритетним. Розширення асортименту аксесуарів для цих популярних категорій створить можливості для допродажів та підвищить середній чек.

*   **Масштабування успішної регіональної моделі.** Аналіз показав, що штат **Нью-Йорк (NY)** генерує **65.74%** доходу всієї мережі. Це підтверджує ефективність магазину в м. Болдвін. Рекомендується впровадити аналогічні методи управління запасами та маркетингові підходи в магазинах Техасу та Каліфорнії для виправлення їх низьких показників (менше **24%** разом).

*   **Стратегія утримання клієнтів.** Попри загальні обсяги продажів, рівень повернення клієнтів становить лише **15.19%**. Оскільки залучення нового покупця коштує дорожче, ніж утримання існуючого, розвиток програми лояльності та персоналізований маркетинг для тих **85%** клієнтів, що здійснили лише одну покупку, є критично важливим для стабільного довгострокового доходу.

*   **Використання сезонності для управління ланцюгом постачання.** Виявлений пік продажів у **березні-квітні** (особливо рекордний квітень 2018 року) вказує на необхідність підготовки складських запасів заздалегідь. Це допоможе уникнути дефіциту товарів у пікові періоди та забезпечить безперебійне обслуговування попиту, що безпосередньо впливає на лояльність клієнтів.

*   **Перегляд неефективних активів.** Категорії з низьким рівнем продажів, як-от дитячі велосипеди, потребують ретельного аналізу. До впровадження спеціальних акцій, вони лише займають складські площі. Мій аналіз дозволяє виділити ці "слабкі місця" та переспрямувати кошти у найбільш прибутковіші товари.

## 7. Наступні кроки

### Сегментація клієнтів
Розділити покупців на групи: «новачки», «постійні» та «ті що йдуть» щоб розуміти, кому давати знижки.

### Аналіз спільних покупок
З’ясувати, які товари люди зазвичай беруть разом наприклад велосипед + шолом щоб пропонувати корисні доповнення.

### Прогноз продажів
Спрогнозувати, скільки велосипедів буде продано наступного сезону, щоб заздалегідь замовити потрібну кількість товару на склад.

### Оцінка ефективності реклами
Порахувати скільки прибутку йде на маркетинг, щоб уникати неефективних реклам.
