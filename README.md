# Аналіз продажів мережі веломагазинів (SQL, Excel, Tableau)

Цей проект присвячений комплексному аналізу даних мережі веломагазинів. Я пройшов шлях від підготовки даних за допомогою SQL до створення фінальних бізнес-дашбордів.

---

## 🛠 Технології та інструменти
- **SQL (PostgreSQL/MySQL):** Написання складних запитів, робота з віконними функціями та CTE.
- **Microsoft Excel:** Очистка даних, створення зведених таблиць (Pivot Tables) та інтерактивного дашборду.
- **Tableau:** Візуалізація даних та створення інтерактивного звіту.

---

## 📈 Етап 1: SQL-аналітика
У файлі `bikesales-script.sql` я підготував скрипти для вивантаження та аналізу даних.
**Ключові аспекти коду:**
- **Валідація:** Перевірка на NULL-значення для забезпечення якості даних.
- **ТОП-аналіз:** Використання `RANK()` для визначення 3-х найпопулярніших товарів у кожній категорії.
- **Бізнес-метрики:** Розрахунок доходу за місяцями та аналіз активності клієнтів (повторні замовлення).

---

## 📊 Етап 2: Excel Дашборд
Файл: `bikestores-visualizations.xlsx`
Я створив **Executive Dashboard**, який включає:
- Аналіз динаміки доходу (Revenue) по роках та місяцях.
- Розподіл продажів за штатами та конкретними магазинами.
- Рейтинг менеджерів з продажів.
- Використання **Slicers (зрізів)** для інтерактивної фільтрації даних.

---

## 🎨 Етап 3: Візуалізація в Tableau
Оскільки Tableau дозволяє створювати більш глибоку візуалізацію, я виніс аналіз туди.

> **[Натисніть тут, щоб переглянути мій дашборд у Tableau Public](https://public.tableau.com/app/profile/jaroslav.stetsiuk/viz/BakeStoreDashboard/Dashboard1?publish=yes)**

**У Tableau реалізовано:**

- Глибока аналітика категорій та брендів.
- Інтерактивні фільтри для детального дослідження даних.

## 💡 Ключові висновки
- Виявлено сезонність продажів з піком у весняно-літній період.
- Визначено бренди-лідери, які приносять основну частину прибутку.
