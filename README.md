# 🚲 Case Study: Аналіз ефективності продажів мережі BikeStores (SQL, Excel, Tableau)

##  1. Передумови та огляд проєкту
Проєкт присвячений глибокому аналізу діяльності американської мережі магазинів велосипедів. Я дослідив дані за 2016–2018 роки, щоб перетворити суху статистику на конкретні бізнес-рішення.

**Основна мета:**
* Виявити закономірності зростання виторгу та сезонні коливання.
* Проаналізувати продуктовий портфель (бренди та категорії).
* Розробити рекомендації для покращення стратегії утримання клієнтів.

## 4. Executive Summary

### Overview of Findings
Після стабільного зростання у 2016 році, продажі компанії BikeStores досягли свого піку у 2017 році, продемонструвавши приріст виторгу на **66.4%**. Основні показники ефективності (KPI) демонструють високу концентрацію ринку в штаті Нью-Йорк та домінування спортивних категорій товарів. Хоча 2018 рік показав певне коригування обсягів, весняний сезон залишається критичним драйвером доходу, забезпечуючи найбільші сплески активності.

Нижче наведено огляд головної сторінки дашборду Tableau. Повний інтерактивний звіт доступний за посиланням у розділі технічної документації.

### Key Performance Indicators (KPIs):
*   **Total Revenue**: **$8.51M** — Загальний виторг за весь період (2016–2018).
*   **Total Units Sold**: **7,078** — Загальна кількість проданих велосипедів.
*   **Customer Base**: **1,445** — Кількість унікальних клієнтів.
*   **Retention Rate**: **15.19%** — Частка клієнтів, що повернулися за повторною покупкою.

### Sales Trends:
*   **Peak Performance**: Продажі компанії досягли максимуму в **квітні 2018 року**, що стало рекордним місяцем за весь період спостережень.
*   **Regional Dominance**: Штат **Нью-Йорк (NY)** є ключовим регіоном, генеруючи **65.74%** усього виторгу мережі через магазин у м. Болдвін.
*   **Category Leaders**: Категорія **Mountain Bikes** забезпечує майже половину доходу компанії (**$1.33M**).
*   **Brand Strength**: Бренди **Trek** та **Electra** сукупно контролюють понад **68%** ринку продажів мережі.

## 5. Глибоке занурення у висновки (Deep Dive)

У цьому розділі я розкриваю логіку аналізу: від виявлення бізнес-проблеми до її технічного підтвердження за допомогою SQL та фінальної візуалізації в Tableau.

### А. Сезонність та пікові навантаження
**Проблема:** Необхідність точного прогнозування складських запасів для запобігання дефіциту під час сплесків активності.
**Аналіз:** Вивчення продажів за 2016–2018 роки показало, що серпень та вересень є критично важливими місяцями. Особливу увагу привертає вересень 2018 року — зафіксовано аномальний пік продажів.

*   **Технічне підтвердження:** Для отримання цих цифр я використав SQL-агрегацію (номер 12 у скрипті[SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
-- Аналіз доходів на місяць для виявлення сезонності
SELECT

GROUP BY year, month
ORDER BY year, month;
```
### Б. Продуктова стратегія та лідери ринку

**Проблема:** Визначення ключових сегментів для фокусування маркетингових зусиль та оптимізації складських запасів на основі прибутковості категорій.

**Аналіз:** Дані візуалізації підтверджують, що категорія **Mountain Bikes** є головним драйвером прибутку, забезпечуючи **48,97%** усього виторгу ($1,32 млн). Наступними за важливістю сегментами є **Cruisers** та **Cyclocross**, які сукупно дають майже **30%** доходу. Найнижчу ефективність демонструють **дитячі велосипеди** (лише **3,62%**), що вказує на низьку рентабельність та доцільність перегляду цього напрямку..

*   **Технічне підтвердження:** Для виділення ТОП-3 товарів у кожній категорії я застосував віконну функцію `RANK()` (номер 13 у SQL-скрипті[SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
-- Ранжування товарів усередині кожної категорії за кількістю продажів
WITH top_3 AS (
    SELECT category_name, product_name, total_units,
    RANK() OVER (PARTITION BY category_name ORDER BY total_units DESC) AS rnk
    FROM "BikeStores.xlsx - Data"
)
SELECT category_name, product_name, total_units
FROM top_3 
WHERE rnk <= 3;
```
### В. Утримання клієнтів (Customer Retention)
**Проблема:** Висока вартість залучення нових клієнтів при низькій лояльності (більшість покупців не повертаються за повторною покупкою).
**Аналіз:** Розрахунок показника Retention показав, що понад **85% клієнтів** здійснюють лише одну покупку[span_0](start_span)[span_0](end_span). *   **Технічне підтвердження:** Я розрахував відсоток клієнтів, що повернулися, за допомогою CTE та агрегатних функцій (номер 15 у SQL-скрипті [SQL (PostgreSQL/MySQL)](./bikesales-script.sql)):
```sql
-- Розрахунок відсотка Retention (клієнти, що зробили більше 1 замовлення)
WITH customer_orders AS (
    SELECT customers, COUNT(order_id) AS total_orders
    FROM "BikeStores.xlsx - Data"
    GROUP BY customers
)
SELECT 
    ROUND(SUM(CASE WHEN total_orders > 1 THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS returning_percentage
FROM customer_orders;
```

## 💡 6. Recommendations (Бізнес-цінність та рекомендації)

На основі виявлених інсайтів було сформовано наступні рекомендації, що мають на меті підвищення прибутковості та ефективності BikeStores:

*   **Оптимізація товарного портфеля для максимізації доходу.** Оскільки бренди **Trek** та **Electra** генерують понад **68%** усього виторгу, а категорія **Mountain Bikes** приносить майже половину прибутку (**$1.33M**), фокусування на цих сегментах є пріоритетним. Розширення асортименту аксесуарів для цих популярних категорій створить можливості для допродажів (upsell) та підвищить середній чек.

*   **Масштабування успішної регіональної моделі.** Аналіз показав, що штат **Нью-Йорк (NY)** генерує **65.74%** доходу всієї мережі. Це підтверджує ефективність поточної бізнес-моделі магазину в м. Болдвін. Рекомендується впровадити аналогічні методи управління запасами та маркетингові підходи в магазинах Техасу та Каліфорнії для виправлення їх низьких показників (менше **24%** спільно).

*   **Стратегія утримання клієнтів як драйвер зростання.** Попри загальні обсяги продажів, рівень повернення клієнтів становить лише **15.19%**. Оскільки залучення нового покупця коштує дорожче, ніж утримання існуючого, розвиток програми лояльності та персоналізований маркетинг для тих **85%** клієнтів, що здійснили лише одну покупку, є критично важливим для стабільного довгострокового доходу.

*   **Використання сезонності для управління ланцюгом постачання.** Виявлений пік продажів у **березні-квітні** (особливо рекордний квітень 2018 року) вказує на необхідність підготовки складських запасів заздалегідь. Це допоможе уникнути дефіциту товарів у пікові періоди та забезпечить безперебійне обслуговування попиту, що безпосередньо впливає на лояльність клієнтів.

*   **Перегляд неефективних активів.** Категорії з низьким рівнем продажів, як-от дитячі велосипеди, потребують ретельного аналізу. До впровадження бандлів (комплектів) або спеціальних акцій, вони лише займають складські площі. Мій аналіз дозволяє виділити ці "слабкі місця" та переспрямувати оборотні кошти у найбільш маржинальні товари.
